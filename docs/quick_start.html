Quick Start Guide¶

Dependencies¶

The package requires numpy, scipy, pandas and TensorFlow.

The package versions this package has been tested on is shown in the following table.

Package Version Annotation
numpy 1.15.2  
scipy 1.1.0  
pandas 0.23.4  
TensorFlow 1.10.1  

A virtual environment manager like Conda is recommended.
